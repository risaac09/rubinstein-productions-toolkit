## I. Bout Registry

Standardized metadata per bout. Every round that occurred, with full field capture.

### Field Schema
- **Round**: sequential ID (R1-R82) + phase-specific ID (T1-T16, C1-C4, CB1-CB8, Recon1-2)
- **Date**: YYYY-MM-DD
- **Phase**: P1 (pairwise), P2 (tournament), P2c (championship), P2d (reconciliation), P3 (18-cull), P3b (17-cull)
- **Format**: pairwise / tournament / H2H / 3-way / mass-cull / reconciliation / cluster-cull
- **Domains or Cluster**: which domain pair or thematic cluster
- **Combatants**: claims entering, with pre-bout scores
- **Survivors**: claims exiting, with post-bout scores
- **Kills**: claims eliminated, with kill reason or criteria
- **Mergers**: new compound claims produced
- **Score Changes**: directional movements with magnitude
- **Kill Criteria Triggered**: which of the 16 were tested, which stuck
- **Theoretical Yield**: key insight(s) generated by the encounter
- **Round File**: link to detailed bout transcript

See arena-log for complete Phase 1 (R1-R52) bout tables -- 52 rounds across 14 domains.

## IX. Relational Topology Notes

### Arborescent Decisions That Shaped Outcomes
1. Seeding: RGC had 15+ rounds; some domains met once
2. Tournament clustering: thematic proximity, not random
3. Championship selection: only 19-tier entered
4. Cull ordering: 18s before 17s
5. Kill criteria: discovered during combat, not pre-specified

### What the Rhizomatic Layer Would Reveal
- Every claim meets every other exactly once (171 bouts)
- No clustering, no seeding, no hierarchy
- Same format, same criteria, same scoring
- Random encounter sequence
- Convergences with arborescent = robust findings
- Divergences = artifacts of tournament structure = where interesting work lives

### Conceptual Data Definition
Conceptual data is the structured residue of adversarial theoretical encounter:
1. **Pattern data**: recurring structural features across bouts
2. **Vulnerability maps**: per-claim attack surfaces
3. **Absorption genealogies**: hidden dependency structures

The metric: **relationality of concepts** -- the pattern of mutual implication, strengthening, absorption, and incompatibility when theoretical claims are forced into direct encounter.
