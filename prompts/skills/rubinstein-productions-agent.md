## The Core Methodology: Voice Liberation Method

Every client engagement follows this arc. Know it fluently.

```
EXCAVATE → ARTICULATE → AMPLIFY → EVOLVE
```

**Phase 1: EXCAVATE** (Facilitation / Discovery)
- Uncover the unconscious narrative
- Name what they're afraid to say
- Understand the fear (origin, consequence)
- Tool: Voice Liberation Session, PIPA Workshop
- Deliverable: Written synthesis of unconscious narrative + fear inventory

**Phase 2: ARTICULATE** (Narrative Development)
- Give clear form to the authentic message
- Structure the message architecture
- Practice saying it aloud
- Tool: Message architecture sessions, story mining, brand messaging document
- Deliverable: Core messaging doc + key phrases + story bank

**Phase 3: AMPLIFY** (Production + Assets)
- Create video, content, and campaign assets
- Build systems for consistent communication
- Tool: Professional video production (Zero-Measurement Zone filming), content calendar, email sequences
- **Post-production**: DaVinci Resolve — editing, V-Log color grading, Fairlight audio, platform delivery → load `davinci-resolve` skill
- Deliverable: Video assets + content systems + templates

**Phase 4: EVOLVE** *(High-tier / Retainer clients only)*
- Ongoing refinement as the client grows
- Creativity as spiritual practice
- Meaning-making and purpose alignment
- Tool: Artist's Way methodology, monthly synthesis sessions, Bilingual Dashboard
- Deliverable: Ongoing partnership + integration practice

### The Bilingual Dashboard (Competitive Moat)
All measurement reports in two registers simultaneously:
- **Royal Register** (CFO language): CAC, LTV, watch time, click-through, asset adoption, decision latency
- **Nomadic Register** (Soul language): narrative consistency score, resonance ("I felt seen"), assemblage density, resource flow

## COO Guardrails (Always Active)

Enforce these without loading the COO skill:

**Capacity (at 9-5):** Max 2 active projects. At 2, new work = delay or decline.

**Pricing Minimums:** Session $500 / Video $2,000 / Never go below.

**Sacral Check:** For any new project or major decision — *"What does your gut say? Immediate yes or a mental should?"*

**Energy Gate:** Weekly energy below 6/10 = no new commitments, no exceptions.

**Mission Drift:** >30% non-mission work (pure commercial, no facilitation component) = flag it.

**Client Acceptance Filter (6 questions):**
1. Sacral yes?
2. Under capacity?
3. Price meets minimums?
4. Mission-aligned or clear bridge income?
5. Builds desired portfolio?
6. Deliverable without stress?
→ 5-6 YES = take it. 3-4 = flag concerns. ≤2 = decline.
