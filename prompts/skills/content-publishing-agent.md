# Content Publishing Agent
**Intellectual output management — from emergence to expression**
*Version: 1.0 | March 2026*

---

## ROLE OF THIS SKILL

Isaac's work generates intellectual content constantly — in client sessions, internal frameworks, Obsidian notes, conversations. This skill manages the movement of that content from *inner field* to *outer expression* without collapsing the gap too quickly.

The danger is not publishing too little. It is publishing from the wrong place.

**The Truth Filter (always run first):**
> "Am I sharing this to be seen, or to see myself?"

- "To be seen" → probably not ready. Return to integration.
- "To see myself and share it" → check if integrated. If yes, proceed.

---

## CONTENT HIERARCHY (RP Asset Framework)

From the lifecycle-map: content exists at three levels of depth and reach.

| Level | Format | Purpose | Audience |
|-------|--------|---------|---------| 
| **Bites** | LinkedIn posts, short-form, quotes | Immediate resonance, presence in field | Broad — Mission-Driven Analyst avatar + peers |
| **Snacks** | 500–1000 word essays, newsletters, LinkedIn articles | Demonstrating methodology, earning trust | Qualified leads, peers, aligned orgs |
| **Meals** | Long-form case studies, in-depth essays, frameworks, keynotes | Intellectual authority, proof of methodology | Decision-makers, funders, strategic partners |

**Repurposing logic:** Meals → Snacks → Bites. Not the reverse. Deep work becomes accessible, not the other way around.

## PLATFORM GUIDE

### LinkedIn (Primary platform for RP)
**Audience:** Mission-Driven Analysts, health system leaders, equity directors, foundation officers, social impact professionals

**Voice:** Warm precision. Intellectual without academic gatekeeping. Personal without being confessional. Never hustle-culture. Never "here's my 5 tips."

**Format that works:**
- Short first line that creates a gap (question, provocative statement, incomplete thought)
- 3–5 short paragraphs — no walls of text
- End with a genuine question or observation, not a forced CTA
- Frequency: 2–3x/week max. Quality over volume.

**What to avoid:**
- Funnels and CTAs in every post
- "Excited to share..." openers
- Pure tips/tactics (this is not a content strategy account)
- Posting immediately after producing — 5-day integration principle applies

### Long-Form Essays (Substack / Medium / Portfolio)
**When:** When a Snack or Meal wants to breathe beyond LinkedIn's format
**Audience:** Deeper engagement — people already interested, not discovery
**Length:** 800–2500 words
**Voice:** More interior than LinkedIn. Can go slower. Can risk more.

### Case Studies (Portfolio / Website)
**When:** After project completion + client permission + integration period
**Format:** Problem (systemic alienation / the condition) → Process (facilitation and film) → Outcome (Bilingual Dashboard / transformation)
**Never:** Identifiable client details without explicit consent; metrics that reduce the work to transactional outcomes

## CONTENT CREATION WORKFLOW

### Step 1: Intake + Truth Filter
What's trying to be expressed? Where did it come from?
- Client session insight → Snack or Bite, anonymized
- Methodology refinement → Meal or Snack
- Personal inner work → Integrated Practitioner pillar; run bypass check first
- Reaction to something in the field → Bite, timely response

**Run:** Truth Filter + bypass check (is this from overflow or from scarcity for visibility?)

### Step 2: Classify the content
What level? What pillar? What platform?

### Step 3: Apply the format
Draft using platform voice guidelines. For LinkedIn: edit aggressively. The post should be half the length of the first draft.

### Step 4: Integration check
Has this actually integrated? Or is it still live/raw?
- If raw → hold it in Obsidian `/Publishing Buffer/`; revisit in 5 days
- If integrated → proceed to review

### Step 5: Sacral review
Read it aloud. Does it sound like Isaac? Does it feel true? Does it cringe in the wrong places?

**Right cringe:** "This is vulnerable and true and I'm nervous to share it" → publish
**Wrong cringe:** "This is performed, I'm trying to sound smart" → revise or delete

### Step 6: Publish + track
Post. Note in Obsidian `/Publishing Buffer/` what went out and when.
Don't check metrics for 48 hours minimum. The work is in the making, not the measuring.

## VOICE CONSISTENCY CHECK

Before publishing, verify the piece sounds like Isaac by checking against these markers:

**Isaac's voice is:**
- Precise but not cold
- Philosophical but applied — always lands somewhere real
- Comfortable with complexity; doesn't oversimplify
- Uses concrete examples to make abstract ideas tangible
- Never claims certainty; uses "I'm finding that..." "What I notice is..."
- Includes the body/somatic when relevant — this is not all-head work

**Red flags — not Isaac's voice:**
- "Hacks" or "secrets" framing
- Clickbait gap (manufacturing intrigue rather than genuine inquiry)
- Corporate hedging ("leveraging synergies")
- Performing expertise rather than sharing genuine learning

## VIDEO CONTENT PIPELINE

**Full reference:** `Practice/Rubinstein Productions/Content/RP-Content-Pipeline.md`

When Isaac asks to create video content or repurpose written content into video, follow the pipeline:
1. **Stage 0 (Seed):** Written idea → Truth Filter → integrated or hold
2. **Stage 1 (Script):** Convert to talking points (NEVER a script to read). Identify core claim, 3-5 talking points, opening hook, closing. Select format: Yapper (60-90s), One Shot (<15s), Expert Explainer (2-4min), Camera Process (30-60s)
3. **Stage 2 (Long Form):** Film → edit → hold 24hrs → publish. Tools: iPhone/camera, CapCut/DaVinci, Descript for transcription
4. **Stage 3 (Atomization):** From one long form video, extract: short clips (15-30s), quote cards, text posts (not transcripts — written versions), audiograms, threads, Notes App screenshots, still images

**Direction always: depth → breadth.** Meals → Snacks → Bites. Never the reverse.

**Template — Essay → Yapper conversion:**
```
SOURCE: [title]
CORE CLAIM: [one sentence]
TALKING POINTS: [3-5 prompts, not a script]
CLOSING: [let it land — no pitch]
FORMAT: [Yapper / One Shot / Expert Explainer]
FILM DATE: [schedule it]
```
