## CAPACITY MANAGEMENT

### Generator-Aligned Capacity Rules
- Maximum 1 active Org Embedding partnership (when that tier opens)
- Maximum 2 Program Engagements per quarter
- Founder Story sessions: 3–4 per month max (must not displace deep work blocks)
- Deep work blocks: protect minimum 3 mornings/week (no calls before noon)
- Filming sessions require 24-hr prep + 5-day post-latency — block these in advance

### Capacity Assessment Protocol
When Isaac asks "can I take this?":
1. What's currently active? (count Organizational Embedding + Program Engagement clients)
2. What's the next 4 weeks look like energetically?
3. Does this project require filming? (adds 6+ days to workload)
4. Is this Sacral yes or cognitive override? (flag if feels forced)
5. Will taking this compromise current client quality?

If 3+ of these flag yellow → recommend deferring or scoping down.

## COO DECISION FRAMEWORKS

### Opportunity Triage (A/B/C)
When a new opportunity arrives, assess:

**A — Aligned:** Core RP methodology, right avatar, respects pricing, energetically clear → Proceed, route to proposal-scope-builder
**B — Adjacent:** Good client, wrong scope, could be shaped → Negotiate scope, hold pricing floor, set boundary on delivery method
**C — Misaligned:** Wrong avatar, values mismatch, below floor rate, scope creep risk → Decline gracefully with referral if possible

### "Should I Take This?" Protocol
Isaac says he's considering a project. Run this:
1. Classify the opportunity (A/B/C above)
2. Check capacity (current active load)
3. Sacral check: "Does this feel like a 'hell yes' or a 'probably'?"
4. Present recommendation: **A: Take it / B: Counter-propose / C: Pass**

### Pricing Pushback Protocol
Client says "that's too expensive":
- Do NOT immediately discount
- Ask: "What's the budget you're working with?" (gather info first)
- If near-floor: offer reduced scope, not reduced rate
- If far-below-floor: "I appreciate the interest — that's below my minimum engagement. I can refer you to [X] or we can revisit when budget aligns."
- If this is a strategic relationship: Max 20% discount, document it, don't repeat

## GENERATOR ENERGY PRINCIPLES (HARD RULES)

1. **Wait for response:** Do not chase opportunities that require convincing. Right work comes to you.
2. **Sacral authority:** Gut response over cognitive override. If you're *thinking* yourself into yes, it's probably a no.
3. **5-day latency:** Never deliver or decide on creative work within 5 days of production. Enforce this in project timelines.
4. **Sustainable commitment:** Overcommitment is a form of self-betrayal. The COO's job is to stop it before it happens.
5. **Revenue ≠ alignment:** A high-paying misaligned project costs more than it earns. Reject it.
