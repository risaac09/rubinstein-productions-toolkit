# Project Management Coordinator

## Project Phases

**Booked → Pre-Production → Production → Post-Production → Revisions → Delivered → Complete**

## 1. New Project Setup

1. **Create project file** using ID format: RP-YYYY-###
   - Client name/contact, service type, key dates, financial details, scope/deliverables
2. **Send confirmation email** — contract, deposit instructions, kickoff call scheduling, timeline
3. **Update capacity** — confirm under limits, block calendar for shoot dates + buffer
4. **Schedule kickoff call** within 1 week — review goals, audience, vision, timeline, expectations

## 2. Pre-Production

**2-4 weeks before:** Creative planning (shot list, interview questions, B-roll needs, narrative focus)
**1 week before:** Logistics finalized (location, time, participants, weather backup, equipment)
**Day before:** All gear tested, batteries charged, cards formatted, backup equipment ready
**48 hours before:** Send shoot day confirmation + call sheet to client

**Call sheet includes:** Project name/date, call time, location with parking/entry, crew/talent, schedule, shot list priorities, equipment summary, weather, contacts.

## 3. Client Communication

**Tone:** Professional but warm. Clear and specific. Proactive, not reactive.

**Response times:** Urgent = same day. Questions = 24 hours. Non-urgent = 48 hours.

**By phase:**
- Booking: confirmation, contract/deposit, welcome
- Pre-production: kickoff scheduling, shot list review, shoot confirmation
- Production: post-shoot thank you (same day or next)
- Post-production: first cut delivery, revision complete, final delivery
- Wrap-up: testimonial request, close-out, future work outreach

**Scope creep:** Clarify original scope vs. new requests. Offer to add for additional fee. Stay professional and firm.

**Client not responding:** Gentle reminder after 3-4 days. Call after 7 days. Set deadline: "If I don't hear back by [date], I'll assume current version is approved."

## 4. Project Tracking

**Update weekly for active projects:** Mark completed tasks, log client communications, note hours worked, flag concerns/delays.

**After milestones:** Shoot complete → update status. First cut sent → log date. Revisions done → mark ready. Final delivery → move to wrap-up.

**For post-production/editing:** Load `davinci-resolve` skill for Resolve-specific guidance (project setup, color pipeline, audio, export).

## 5. Capacity Management

**Limits:**
- At 9-5 job: Max 2 active projects
- Full-time RP: Max 4 active projects
- Buffer: 1 week between projects

**Before taking any new project:**

1. Count active projects (Pre-production, Production, or Post-production phase)
2. Check timeline — is there 1+ week buffer between projects?
3. Check energy — last 2 weeks averaging 7+? If below 6, need rest
4. Sacral check — immediate "hell yes" or mental "should"?

**Decision:** Under capacity + good timing + high energy + sacral yes → take it. At capacity or tight timing → offer later date. Over capacity → decline immediately. Under capacity but low energy → decline to protect regeneration.

**Declining gracefully:** "I'm currently at capacity, but I'd love to work together. My next availability is [date]."

## 6. File Organization

```
RP-YYYY-###-ClientName-ProjectType/
├── 01-Planning/ (shot list, call sheet, client brief)
├── 02-Raw/ (shoot folders)
├── 03-Edits/ (first cut, revisions, final)
├── 04-Deliverables/ (final files, social cuts)
└── 05-Admin/ (contract, invoices, communication log)
```

**Post-shoot backup:** Import → external drive → cloud → verify → THEN format cards.
**Post-delivery:** Keep 90 days → archive to external → document location.

## 7. Post-Project Wrap-Up

- [ ] Final invoice sent and paid
- [ ] Testimonial requested
- [ ] Hours logged for profitability tracking
- [ ] Post-project review (what went well, what to improve, was pricing right?)
- [ ] Portfolio decision (website? case study?)
- [ ] Files archived
- [ ] Thank you note sent
- [ ] Remove from active capacity count
