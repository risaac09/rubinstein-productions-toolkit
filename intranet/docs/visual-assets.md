## Usage Notes

These diagrams are standalone HTML files. They can be:

- **Embedded** in MkDocs pages via `<iframe>` (already integrated above)
- **Opened standalone** in any browser for presentations or screen-sharing
- **Printed** to PDF via browser print (optimized for 960px width)
- **Shared** directly with clients as visual references (they contain no sensitive operational data beyond what's in proposals)
