# Equipment & Logistics

## Camera-Shipping Model

The camera-shipping model is core to RP's methodology. Instead of sending a crew, we ship a camera kit to the participant. They film on their own terms, in their own space, without surveillance pressure. This produces more authentic footage because the participant is the author, not the subject.

### Kit Contents

| Item | Purpose | Notes |
|------|---------|-------|
| iPhone + case | Primary camera | Pre-configured: resolution, frame rate, filming app installed |
| Tripod / adjustable mount | Stability | Works on tables, shelves, floor. No special skills needed. |
| Lavalier microphone | Audio capture | Clip-on, plugs directly into phone |
| Battery pack | Extended power | Fully charged before shipping. 8+ hours backup. |
| SD card + adapter | Footage transfer | Include written transfer instructions |
| Camera Kit Guide | Client reference | `06-camera-kit-guide.docx` — printed and included in kit |
| Return shipping label | Kit return | Pre-paid. Include carrier name and drop-off instructions. |

### Shipping Protocol

1. **5 days before filming window:** Ship kit via tracked carrier
2. **Confirm delivery:** Text or email participant when tracking shows delivered
3. **Quick orientation:** 15-minute phone/video call walking through the kit (optional but recommended)
4. **During filming window:** Available by text for technical questions
5. **After filming:** Participant ships kit back using pre-paid label
6. **Upon receipt:** Confirm return, download and back up all footage immediately

### Footage Management

| Step | Action | Location |
|------|--------|----------|
| Ingest | Download from SD card, verify all files | Local SSD |
| Backup | Copy to second drive | External backup |
| Organize | Name files: `[Client]_[Date]_[Participant]_[Take#]` | Project folder |
| Review | Watch all footage, log selects | DaVinci Resolve |
| Archive | Move to long-term storage after delivery | Archive drive |
| Delete | After client confirms receipt + 90 days | All locations |

!!! warning "Data Return Policy"
    All footage belongs to the client. After final delivery and 90-day confirmation window, delete all copies from RP drives. Retain nothing.

## Production Equipment

### Camera

Primary: iPhone 15 Pro (or current equivalent)

- 4K/24fps for cinematic feel
- ProRes recording for maximum edit flexibility
- Filmic Pro app for manual controls
- Always shoot in landscape unless client requests otherwise

### Audio

- Rode SmartLav+ (lavalier, wired)
- Rode Wireless GO II (wireless, for movement)
- Always prioritize audio quality over video quality — audiences forgive visual imperfection but not bad sound

### Lighting

- For shipped kits: natural light only (instruct participant to face window)
- For on-site sessions: Aputure MC (compact LED, adjustable temperature)
- Key principle: warm, soft, directional. Never overhead fluorescent.

### Post-Production

- DaVinci Resolve Studio — editing, color grading, audio (Fairlight)
- V-Log color profile for maximum dynamic range
- Standard export: H.264, 4K, high bitrate for master; 1080p for web delivery

## Facility Requirements

### For On-Site Sessions

- Quiet room with window light
- Enough space for 2 chairs facing each other at 45-degree angle
- No overhead fluorescent lighting (turn off if possible, use portable LED)
- Temperature comfortable (people tense up in cold rooms)
- Water available for participant
- Door that closes (interruptions break the container)

### For Remote Sessions

- Zoom or Google Meet (participant's preference)
- Ask participant to use headphones (audio quality)
- Request they sit near a window
- Record locally on participant's end if possible (better quality than cloud recording)
